# oligula-automatizacoes

Desktop automation app made with React, TypeScript & Rust

## Requirements

- Rust 1.78 or higher
- [Tauri prerequisites](https://tauri.app/v1/guides/getting-started/prerequisites)
- pnpm 9 or higher
- Node.js 20 or higher

## Using

### Setup

- `pnpm install`

### Running in Development

- `pnpm tauri dev`

### Building

- `pnpm tauri build`
